Question-1: What are the different ways to select an element in the DOM?

1 getElementById(): Selects an element by its id. 2 getElementsByClassName(): Selects elements by their class name.

3 getElementsByTagName(): Selects elements by their tag name. 4 querySelector(): Selects the first element that matches a CSS selector.

5 querySelectorAll(): Selects all elements that match a CSS selector. 6 getElementsByName(): Selects elements by their name attribute.

Question-2: What is the difference between innerHTML, innerText, and textContent ?

1 innerHTML allows reading and modifying HTML content.

2 innerText returns the visible text only, ignoring hidden text and HTML tags.

3 textContent returns all text, including hidden text, and is generally faster.

Question-3: What is event delegation in the DOM?

Event delegation is a technique where a single event listener is added to a parent element to handle events for its child elements. When an event is triggered on a child element, it bubbles up to the parent, and the parent handles the event, typically using the event.target to identify the specific child element. This improves performance and makes it easier to manage events for dynamically added elements.

Question-4: What is event bubbling in the DOM?

Event bubbling is the process in which an event, triggered on an element, propagates (or "bubbles") up through its parent elements in the DOM hierarchy. The event starts from the target element and moves upward, triggering any event listeners on the parent elements along the way, until it reaches the root of the document.

This allows parent elements to listen for events on their child elements without needing to attach individual listeners to each child.
